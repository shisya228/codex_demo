# 数独求解器（Python + Numba）

这是一个使用 **Python + Numba** 的数独求解器项目，支持从命令行读取题目并输出解。

## 安装

```bash
pip install -e .
```

## 使用方式

将题目以 81 位数字/点号输入（0 或 . 表示空格），例如：

```bash
solve-sudoku "530070000600195000098000060800060003400803001700020006060000280000419005000080079"
```

也可以从文件读取：

```bash
solve-sudoku --file puzzle.txt
```

## 输入格式

- 81 个字符
- `0` 或 `.` 表示空格
- 其余数字为 `1-9`

## 说明

- 核心求解采用回溯算法，并用 Numba JIT 加速。
- 默认会返回第一个解。
